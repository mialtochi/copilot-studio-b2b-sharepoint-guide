# Sample Configuration Files

This directory contains example configuration files and reference settings for the B2B SharePoint Copilot agent.

## Files in This Directory

### 1. `agent-config-example.json`
Sample Copilot Studio agent configuration settings.

### 2. `sharepoint-permissions-example.csv`
CSV template for bulk assigning SharePoint permissions.

### 3. `entra-id-groups-example.csv`
CSV template for managing Entra ID group memberships.

## How to Use

1. **Review the sample files** to understand structure
2. **Customize** with your organization's details
3. **Use templates** for bulk operations where applicable
4. **Reference** when troubleshooting configuration issues

## Important Notes

- **Never share** files containing real user emails or sensitive data
- **Test in non-production first** before bulk operations
- **Backup** existing configurations before making bulk changes
- **Verify** all customizations before deploying
